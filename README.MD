# 🎓 Student Management System (Java + JDBC + MySQL)

## 📌 Project Description
This is a **Student Management System** developed using **Java, JDBC, and MySQL**.  
The application allows users to manage student records efficiently through basic database operations.

---

## 🚀 Features
- ➕ Add new student details  
- 📋 View all student records  
- ✏️ Update existing student information  
- ❌ Delete student records  
- 🔍 Search student by ID  

---

## 🛠️ Technologies Used
- **Java**
- **JDBC (Java Database Connectivity)**
- **MySQL**
- **OOP Concepts**

## ⚙️ How to Run the Project

1. Clone the repository:
```
git clone https://github.com/aishwaryasuryawanshi204-debug/student-management-system
```

2. Open the project in your IDE (Eclipse / IntelliJ)

3. Configure MySQL database:
- Create a database (e.g., `studentdb`)
- Create a table for students

4. Update database credentials in code:
```
String url = "jdbc:mysql://localhost:3306/studentdb";
String username = "root";
String password = "your_password";
```

5. Run the `Main.java` file
